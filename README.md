# Employee Attrition Prediction using AI

## Project Overview

This project demonstrates how Artificial Intelligence can be used in Human Resource Management to predict employee attrition. Employee attrition occurs when employees leave an organization, which can cause productivity loss and increased recruitment costs.

Using Machine Learning, this project analyzes HR data such as age, salary, job satisfaction, work hours, and years at the company to predict whether an employee is likely to leave.

The project is implemented using **Python and Google Colab**.

---

## Objective

The objective of this project is to:

* Predict employee attrition using HR analytics
* Identify employees who may leave the organization
* Support HR departments in making data-driven decisions

## Dataset Features

The dataset contains the following columns:

| Feature         | Description                                            |
| --------------- | ------------------------------------------------------ |
| Age             | Age of the employee                                    |
| Salary          | Monthly salary                                         |
| JobSatisfaction | Employee job satisfaction level (1-5)                  |
| WorkHours       | Average work hours per week                            |
| YearsAtCompany  | Number of years the employee has worked in the company |
| Attrition       | 1 = Employee left, 0 = Employee stayed                 |

## Machine Learning Model

The project uses **Logistic Regression** to predict whether an employee is likely to leave the company.

The model is trained using HR dataset features and then used to predict attrition for new employees.

---

## Example Output

The trained model predicts:

* **1 → Employee likely to leave**
* **0 → Employee likely to stay**

This helps HR teams take preventive actions such as improving employee satisfaction or offering incentives.
